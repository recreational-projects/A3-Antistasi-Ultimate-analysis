# A3-Antistasi-Ultimate-analysis

- Clone source data [SilenceIsFatto/A3-Antistasi-Ultimate](https://github.com/SilenceIsFatto/A3-Antistasi-Ultimate)
- Clone this repo, and preferably set up a virtual environment for it
- Edit `config.toml` to point to source data directory and desired output directory
- Install dependencies: `pip install .` or equivalent
- Run `python main.py` or equivalent
